# Макет интерфейса (проверка ТЗ)

Статический HTML/CSS по [`../FullSpecification.md`](../FullSpecification.md): десктопный UI, все основные блоки и **типы уведомлений** из спецификации.

## Просмотр

Откройте в браузере файл:

`LegalDocInspectorV2/interface/index.html`

(двойной клик или `open interface/index.html` — пути к `css/app.css` и `js/mock.js` относительные).

## Состав

| Файл | Назначение |
|------|------------|
| `index.html` | Разметка: вход, пакет, загрузка, прогресс, конфликты, форма, расчёт, скачивание, справочник иконок, боковая панель уведомлений, тосты |
| `css/app.css` | Стили |
| `js/mock.js` | Заглушки (без API) |

Полная функциональность **не реализуется** — только визуализация для ревью ТЗ.
