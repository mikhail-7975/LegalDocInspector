Задача прервана. Введите данные вручную; повтор OCR не выполняется. Следующая задача в очереди обрабатывается.
OCR_PIPELINE_CRASH
Лимиты (§2)
Превышено максимальное число комплектов (30) или справок в комплекте (20). Сообщение из config.
VALIDATION_ERROR · LIMIT_EXCEEDED
Валидация загрузки (FR-2)
Договор должен быть в формате PDF; справка — .xls / .xlsx / .xlsm. Тексты из config.
UNSUPPORTED_FORMAT
Размер файла (NFR-10 / открытый Q-3.2.1)
Превышен допустимый размер файла (порог задаётся после уточнения).
FILE_TOO_LARGE
ETA OCR (§5 Q-5.0.2)
Ориентировочное время обработки: ~12 мин (в очереди 5 документов). Формула — на реализацию.
Browser Push (§5)
Разрешите уведомления — оповестим о завершении длинной очереди (email не требуется).
beforeunload (§4 FR-1)
